# Hostel Complaint Management System

A full-stack web application designed to streamline complaint management in hostels. Students can raise and track complaints, while caretakers can manage and resolve them efficiently.

---

## Features

### Student Module
- Secure login and authentication
- Create new complaints
- View own complaints
- Search complaints by room number
- Filter complaints by status and category
- Server-side pagination
- Real-time status tracking

### Caretaker (Admin) Module
- Secure login
- View complaints from all students
- Search, filter, and paginate complaints
- Update complaint status (Open / In Progress / Resolved)
- Dashboard with complaint statistics

---

##  Tech Stack

### Frontend
- React
- Tailwind CSS
- shadcn/ui
- Zod for form validation

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
